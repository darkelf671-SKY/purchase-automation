생성 완료 모델: Gemini 3.1 Flash Lite
ℹ
상태 바 우측에 "모델: Gemini 3.1 Flash Lite" 표시. 설정에서 선택한 모델명이 그대로 반영됨. 한도 초과 시 사용자가 설정에서 모델을 변경하면 즉시 반영.
구현 메모 — dialog_settings.py 변경 포인트
# config.py — 모델 목록 및 기본값 추가
GEMINI_MODEL_OPTIONS = [
  "gemini-2.0-flash",           # 안정적, 범용
  "gemini-2.0-flash-lite",     # 가볍고 빠름
  "gemini-2.5-flash-lite-preview",# 경량 최신
  "gemini-3.1-flash-lite-preview",# 최신 프리뷰
]
GEMINI_MODEL_LABELS = {
  "gemini-2.0-flash": "Gemini 2.0 Flash (안정적, 범용)",
  "gemini-2.0-flash-lite": "Gemini 2.0 Flash Lite (가볍고 빠름)",
  "gemini-2.5-flash-lite-preview": "Gemini 2.5 Flash Lite (경량 최신)",
  "gemini-3.1-flash-lite-preview": "Gemini 3.1 Flash Lite (최신 프리뷰)",
}
GEMINI_DEFAULT_MODEL = "gemini-2.0-flash"

# dialog_settings.py — AI 모델 Combobox 추가
self._ai_model_var = tk.StringVar(value=settings.get("gemini_model", GEMINI_DEFAULT_MODEL))
model_labels = list(GEMINI_MODEL_LABELS.values())
ttk.Combobox(frame, textvariable=self._ai_model_var, values=model_labels,
            state="readonly", width=38).grid(...)

# _on_save() — 선택된 라벨을 모델 ID로 역변환하여 저장
label_to_id = {v: k for k, v in GEMINI_MODEL_LABELS.items()}
settings["gemini_model"] = label_to_id.get(self._ai_model_var.get(), GEMINI_DEFAULT_MODEL)
