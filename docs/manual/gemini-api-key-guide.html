5 프로그램에 API 키 등록 약 30초

복사한 API 키를 구매기안 자동화 시스템에 등록합니다.

1
구매기안 자동화 시스템을 실행합니다.
2
상단 메뉴의 [설정] 버튼을 클릭합니다.
3
설정 창에서 "AI 설정" 섹션을 찾습니다.
설정 ×
── 기존 설정 (산출 폴더, 부서명, 검수자, 입회자) ──

✨ AI 설정

👉 Ctrl+V로 붙여넣기!

※ Google AI Studio에서 무료 발급 | 기안 내용 AI 생성에 사용

4
"Gemini API 키" 입력란을 클릭한 후, Ctrl+V를 눌러 복사한 키를 붙여넣습니다.
5
[저장] 버튼을 클릭합니다.
✅ API 키 등록 완료! 이제 AI 기안 내용 생성 기능을 사용할 수 있습니다. API 키는 한 번만 등록하면 됩니다. 프로그램을 다시 실행해도 유지됩니다.
6 정상 작동 확인 약 1분

API 키가 제대로 등록되었는지 확인해봅니다.

1
구매 조사 탭으로 이동합니다.
2
하단의 "④ 기안 작성" 섹션에서 품명을 입력합니다. (예: "A4용지")
3
"내용 *:" 필드 아래의 ✨ AI 활용하기 버튼을 클릭합니다.
4
AI 기안 생성 다이얼로그에서 간단한 설명을 입력합니다.
예: 사무실 복사용지 부족으로 추가 구매
5
[생성하기] 버튼을 클릭하고 결과가 나타나는지 확인합니다.
✅ 축하합니다! AI가 기안 내용을 생성했다면 모든 설정이 정상적으로 완료된 것입니다!
[적용]을 클릭하면 기안 내용 필드에 자동으로 입력됩니다.
❌ 오류가 발생했다면?
  • "API 키가 올바르지 않습니다" → 키를 다시 복사하여 붙여넣기 해보세요
  • "인터넷 연결을 확인하세요" → Wi-Fi나 랜선 연결 상태를 확인하세요
  • "API 호출 한도에 도달했습니다" → 1분 후에 다시 시도하세요

자주 묻는 질문 (FAQ)

❓ 정말 무료인가요? 나중에 결제되는 것은 아닌가요?
네, 완전히 무료입니다. Google Gemini API의 무료 티어는 신용카드 등록 없이 사용할 수 있습니다. 분당 15회, 하루 1,000회 제한이 있지만 개인 사용에는 충분합니다. 유료로 전환하려면 별도로 Google Cloud 결제 계정을 직접 설정해야 하므로, 자동으로 과금되지 않습니다.
❓ API 키를 잊어버렸어요. 어떻게 하나요?
aistudio.google.com/apikey에 다시 접속하면 기존에 생성한 API 키 목록을 볼 수 있습니다. 키 값 전체가 보이지 않을 수 있는데, 그 경우 기존 키를 삭제하고 새로 만들면 됩니다.
❓ API 키가 유출된 것 같아요. 어떻게 하나요?
aistudio.google.com/apikey에서 해당 키 옆의 삭제(🗑) 버튼을 클릭하여 즉시 삭제하세요. 그 후 "Create API key" 버튼으로 새 키를 발급받고 프로그램 설정에서 교체하면 됩니다.
❓ 공공기관 네트워크에서 작동하나요?
기관의 보안 정책에 따라 외부 API 접속이 차단될 수 있습니다. googleapis.com 도메인으로의 HTTPS(443) 접속이 허용되어야 합니다. 차단된 경우에도 AI 기능 외의 다른 모든 기능(수동 입력, 템플릿 등)은 정상 작동합니다. 네트워크 담당자에게 문의하세요.
❓ AI가 생성한 내용을 그대로 써도 되나요?
AI가 생성한 내용은 초안입니다. 반드시 내용을 확인하고 필요에 따라 수정한 후 사용하세요. 특히 금액, 수량, 날짜 등 사실 정보는 AI가 정확하게 생성하지 못할 수 있으므로 꼭 확인해야 합니다. "적용" 후에도 내용 필드에서 직접 편집할 수 있습니다.
❓ 다른 컴퓨터에서도 같은 API 키를 사용할 수 있나요?
네, 하나의 API 키를 여러 컴퓨터에서 사용할 수 있습니다. 각 컴퓨터의 프로그램 [설정]에서 동일한 API 키를 등록하면 됩니다. 단, 무료 한도(분당 15회)는 모든 컴퓨터의 사용량이 합산됩니다.
❓ EXE 파일로 배포할 때 API 키도 포함되나요?
아닙니다. API 키는 EXE 파일에 포함되지 않습니다. 각 사용자가 직접 Google AI Studio에서 자신의 API 키를 발급받아 등록해야 합니다. 이 가이드를 함께 배포하면 됩니다.
